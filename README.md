# 🚀 GitHub Issue Tracker

A professional, responsive web application designed to help developers monitor and manage GitHub repository issues with ease.

**🔗 Live Demo:** [https://github-issue-tracker-sa5.netlify.app/](https://github-issue-tracker-sa5.netlify.app/)

---

## 📖 Project Overview
This project was built to provide a clean and intuitive interface for tracking issues. It integrates directly with the GitHub environment to display relevant data while ensuring a high-quality user experience across all devices.

## 🛠️ Technology Stack
-**Tech:**JavaScript
- **Styling:** Tailwind CSS
- **Platform:** Netlify (Deployment)

## ✨ Main Features
- **Issue Dashboard:** A centralized view of all repository issues.
- **Responsive UI:** Seamless experience on mobile, tablet, and desktop.
- **Interactive States:** Real-time feedback and smooth transitions.
- **Modern Design:** Dark-themed aesthetic aligned with developer preferences.

## 📦 Dependencies
To ensure smooth performance, this project utilizes:
- `tailwindcss`
- `js`
- `html`
